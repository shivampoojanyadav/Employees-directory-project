# 👨‍💼 Employee Directory App (Django)

Hi 👋, I'm Shivam!

This is an **Employee Directory Web Application** built using Django that helps manage employee records in an organized way.

I built this project while learning how real-world systems handle structured data like employee information, and it gave me a better understanding of CRUD operations and database management.

---

## ✨ Features

* ➕ Add new employees
* 📋 View all employee details
* ✏️ Update employee information
* ❌ Delete employee records
* 🔍 Search employees (optional if implemented)
* 💾 Data stored using SQLite

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS
* **Database:** SQLite

---

## 📂 Project Structure

```bash
env/                # Virtual environment
employee_app/       # Main Django app
templates/          # HTML templates
db.sqlite3          # Database
manage.py           # Django management script
```

---

## ⚙️ How to Run Locally

Follow these steps to set up the project:

```bash
# Clone the repository
git clone https://github.com/shivampoojanyadav/Employees-directory-project.git

# Move into the project folder
cd employee-directory

# Activate virtual environment
.\env\Scripts\Activate

# Install dependencies
pip install django

# Apply migrations
python manage.py migrate

# Run the server
python manage.py runserver
```

Now open 👉 http://127.0.0.1:8000
